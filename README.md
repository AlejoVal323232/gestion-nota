# Sistema de Gestión Escolar

Este es un programa que desarrolle en C++ como una actividad y permite que el usuario ingrese la cantidad de notas a ingresar, las notas y luego calcular el promedio verificando si aprobo o no.

## ¿Qué hace mi programa?

* **Configuración inicial:** Permite definir cuántas notas se van a registrar para adaptar el sistema a cualquier tamaño de grupo.
* **Registro de calificaciones:** Captura cada nota y las guarda en memoria para su posterior análisis.
* **Planilla de resultados:** Genera un reporte automático donde puedo ver qué estudiante aprobó (marcado como [P]) o reprobó (marcado como [F]).
* **Análisis final:** Calcula el promedio general de todo el grupo y da un veredicto final sobre si el conjunto aprobó o no la materia.

## Conceptos que demostré aquí

* **Arreglos:** Uso arreglos para almacenar las notas de forma eficiente.
* **Estructuras de control:** Implementé ciclos `do-while` y `switch-case` para que la navegación sea intuitiva.
* **Validación de estados:** Uso variables booleanas para asegurar que el usuario siga los pasos en orden (primero definir cantidad, luego ingresar notas y al final ver resultados).
* **Formato de consola:** Utilizo herramientas de consola para limpiar la pantalla y presentar los resultados como una tabla organizada.

## Cómo ejecutar este programa

1. Se debe tener instalado un compilador de C++ como Dev-C++.
2. Abrir el archivo `gestion-escolar.cpp` en el editor.
3. Compilar y ejecutar el código.
